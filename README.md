# Zghari

Zghari is a modern portfolio website designed for a professional photographer, showcasing visual storytelling through a clean, elegant, and immersive browsing experience. The website features a curated photography portfolio alongside an integrated booking system, allowing potential clients to explore the photographer's work and schedule sessions through a seamless and intuitive interface.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Built with Claude Code

## Goal
The goal of this project was to create a premium online presence for a photographer by combining a visually driven portfolio with a streamlined booking experience. It demonstrates how thoughtful UI/UX design, responsive layouts, and elegant interactions can help creative professionals present their work, build credibility, and convert visitors into clients while delivering a fast and engaging experience across all devices.
